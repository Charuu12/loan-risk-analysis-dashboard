# 💳 Loan Performance & Credit Risk Analysis Dashboard (SQL + Power BI)

---

## 📊 Overview
This project analyzes loan applications to understand:
- Approval patterns  
- Credit risk  
- Decision-making factors  

using SQL and Power BI.

---

## 🛠️ Tools Used
- **PostgreSQL** (Data Cleaning & Analysis)  
- **Power BI** (Dashboard Visualization)  
- **Excel** (Dataset)  

---

## 🔄 Project Workflow
1. Data Cleaning using SQL  
2. Feature Engineering:
   - Income Groups  
   - Loan Size Categories  
   - Risk Category  
   - Income-to-Loan (ITL) Ratio  
3. SQL Analysis:
   - Loan performance metrics  
   - Credit risk insights  
4. Power BI Dashboard:
   - Executive Overview  
   - Credit Risk Analysis  
   - Decision Intelligence  

---

## 📊 Dashboard Pages

### 🔹 Loan Performance Overview
- Total applications, approval & rejection rates  
- Approval trends by loan size and property area  
- Income group analysis  

---

### 🔹 Credit Risk Analysis
- Credit history vs approval (**key risk driver**)  
- Rejection trends by property area and loan size  
- High-risk applicant segments  

---

### 🔹 Decision Intelligence
- Income-to-Loan Ratio (**affordability analysis**)  
- Scatter plot showing approval behavior  
- Insight: Affordability alone does not determine approval  

---

## 🔍 Key Insights
- Credit history is the **strongest factor** influencing loan approval  
- Larger loan sizes have **higher rejection rates**  
- Only ~28% applicants fall into **high affordability category**  
- Affordability alone does **not fully explain approval decisions**
